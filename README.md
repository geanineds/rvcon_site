# RVCon - Site Responsivo da Empresa de Construção Civil

## Visão Geral

O projeto **RVCon** é o site institucional desenvolvido para a empresa de construção civil RVCon, localizada em Curitiba. O objetivo do site é apresentar a empresa, seus diferenciais, portfólio de obras e facilitar o contato direto com clientes por meio de um atendimento integrado.

## Principais Funcionalidades

* **Página inicial institucional** destacando missão, visão e valores.
* **Portfólio de projetos** apresentando imagens reais das obras entregues.
* **Atendimento rápido** via WhatsApp, com link direto configurado.
* **Chatbot integrado** para suporte inicial ao visitante.
* **Design responsivo**, garantindo navegação fluida em celulares, tablets e desktops.
* **Seções personalizadas** mostrando áreas de atuação da empresa.

## Tecnologias Utilizadas

* **HTML5** para estruturação completa das páginas.
* **CSS3** para o design, responsividade e estilização visual do site.
* **JavaScript** para interatividade, botões dinâmicos e automações.
* **Python** para rotinas internas, testes e scripts auxiliares.
* **Bibliotecas e assets** (imagens, logos e ícones) utilizados no visual.

## Estrutura do Projeto

* `/index.html` – Página inicial.
* `/css/` – Arquivos de estilo.
* `/js/` – Scripts de interação.
* `/assets/` – Logos, ícones e imagens do portfólio.
* `/python/` – Scripts auxiliares do projeto.
